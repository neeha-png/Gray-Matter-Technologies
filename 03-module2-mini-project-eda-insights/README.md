# 3. Module 2 Mini Project — EDA Project with Insights Presentation

**Module:** 2 (Day 12) · **Type:** Mini Project

## Objective
Go beyond Module 1's basic EDA: use advanced visualization (heatmaps, pairplots)
for visual storytelling, and package the result as a presentation, not just a notebook.

## Prerequisite skills
Everything from this module's Day 1–10 (cleaning, feature engineering, EDA
techniques, advanced visualization, pipelines).

## Dataset
Reuse the cleaned dataset from project 2 (`02-module2-case-study-preprocessing`)
so this project is about storytelling, not re-cleaning data.

## Tasks
1. Univariate + bivariate EDA (reuse Module 1 techniques) on the cleaned dataset.
2. Correlation heatmap and a pairplot across key features.
3. Identify 3–5 "interesting" findings (a surprising correlation, a segment that
   behaves differently, a trend over time/category).
4. Turn each finding into one slide/section: chart + one-sentence takeaway.
5. Assemble into a short deck or PDF report (5–10 slides/pages).

## Deliverable
- `notebooks/advanced_eda.ipynb` — analysis behind the findings
- `src/` — any helper plotting functions if reused across charts
- A presentation file (PPTX/PDF/Markdown slides) in this folder, e.g. `INSIGHTS_DECK.md` or `.pdf`

## Done when
You could present the deck to someone unfamiliar with the dataset and they'd
understand the 3–5 findings without needing the notebook.
